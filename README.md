## Features

- Thêm việc cần làm và đánh dấu gạch bỏ khi đã làm xong việc đó
- Lưu danh sách việc cần làm ở local storage

## 📦 Install

```bash
$ git clone https://github.com/kiethohohoho/react-todo-app.git
$ cd todo-app-react
$ npm install
```

### Start client

```bash
$ npm start
```

Client live on: http://localhost:3000/

## 🔨 Build

```bash
npm install
npm run build
```
